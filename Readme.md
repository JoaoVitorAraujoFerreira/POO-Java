# Programação orientada a objetos em Java

Esse projeto é um trabalho avaliativo da minha faculdade para avaliar o conhecimento em POO usando Java.
Ele foi separado em fases, onde cada uma delas emprega um conceito diferente da POO:
- Fase 1
  - Classes e Objetos
- Fase 2
  - Associações e Relacionamentos
- Fase 3
  - Encapsulamento
- Fase 4
  - Herança e Polimorfismo
- Fase 5
  - Interfaces e Abstração
